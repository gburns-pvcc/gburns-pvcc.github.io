Networking & The Internet

Network Classifications
LAN
  • local area network
  • consists of a collection of computers in a single building or building complex
MAN
  • metropolitan area network
  • spans a local community
WAN
  • wide area network
  • links machines over a greater distance
Network Models
bus network
  • in which the machines are all connected to a common communication line
star network
  • in which one machine serves as a central focal point to which all the others are connected
Protocols
CSMA/CD Ethernet
  • Carrier Sense, Multiple Access with Collision Detection
  • This protocol dictates that each message be broadcast to all the machines on the bus
  • Each machine monitors all the messages but keeps only those addressed to itself.
CSMA/CA Wifi
  • Carrier Sense, Multiple Access with Collision Avoidance
  • the policy of trying to avoid collisions rather than trying to detect them
  • when collisions do occur, messages must be retransmitted
Combining Networks
repeater
  • a device that passes signals back and forth between the two original buses (usually with some form of amplification) without considering the meaning of the signals
bridge
  • it connects two buses, but it does not necessarily pass all messages across the connection
  • it looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side
switch
  • a bridge with multiple connections, allowing it to connect several buses rather than just two
  • produces a network consisting of several buses extending from the switch, as spokes on a wheel
router
  • the device that handles the connection between networks to form an internet
  • special purpose computers used for forwarding messages
gateway
  • the "point" at which one network is linked to an internet
  • it serves as a passageway between the network and the outside world.
Client/server model
P2P: peer-to-peer
  • this model involves processes that provide service to and receive service from each other
  • involves processes that execute on a temporary basis
distributed system
  • consists of software units that execute as processes on different computers
cloud computing
  • allows huge pools of shared computers on the network to be allocated for use by clients as needed
Internet Addressing
Tier_1
  • ISPs that consist of very high-speed, high-capacity, international WANs
  • These networks are thought of as the backbone of the Internet
  • are typically operated by large companies that are in the communications business
Tier_2
  • ISPs that tend to be more regional in scope and less potent in their capabilities
  • these networks also tend to be operated by companies in the communications business
Tier_3
  • access ISPs (Internet Service Providers)
  • operated by a single authority that is in the business of supplying Internet access to individual homes and businesses
IP Address
ICANN
  • Internet Corporation for Assigned Names and Numbers
  • a non-profit corporation established to coordinate the Internet's operation
DNS
  • domain name system
  • The hierarchical nature of DNS has been an important factor in its ability to scale as the number of domains, DNS servers, and Internet hosts has grown
TLD
  • Top-level domain
  • .edu/.org/.com/.gov/.mil/.net
Country-code TLDs
  • .au for Australia
  • .ca for Canada
Internet Protocols
FTP
  • File Transfer Protocol
  • for listing and copying files across the network
HTTP
  • Hyper Text Transfer Protocol
  • webpages
SMTP
  • Simple Mail Transfer Protocol
  • defines a way that two computers on the network may interact when transmitting an email message from one host to the other
World Wide Web
URL
  • Uniform Resource Locator
  • contains the information needed by a browser to contact the proper server and request the desired document
client-side activities
  • activities that are performed by a client (such as a browser)
server-side activities
  • activities that are performed by a server (such as a webserver)
Internet four-layer software
application layer
  • consists of those software units such as clients and servers that use Internet communication to carry out their tasks
  • uses the transport layer to send and receive messages over the Internet
transport layer
  • accepts messages from the application layer and to ensure that the messages are properly formatted for transmission over the Internet
  • adds sequence numbers to the small segments it produces so that the segments can be reassembled at the message's destination
  • Then it hands these segments, known as packets, to the network layer
network layer
  • decides in which direction a packet should be sent at each step along the packet's path through the Internet.
  • after determining the proper direction, the network layer hands the packet to the link layer for actual transmission
link layer
  • has the responsibility of transferring the packet
  • the link layer must deal with the communication details particular to the individual network in which the computer resides
Internet is:
  • packet-switching network
  • fault tolerant
  • redundant
Cybersecurity
malware
  • malicious software
virus
  • software that infects a computer by inserting itself into programs that already reside in the machine
worm
  • an autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
Trojan horse
  • a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim
spyware
  • software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
phishing
  • a technique of obtaining information explicitly by simply asking for it
DDoS attack
  • distributed denial of service attack
  • when the work of generating the attacks is distributed over many hosts
Protection
firewall
  • a prevention technique that filters traffic passing through a point in the network
spam filter
  • are firewalls designed to block unwanted email
proxy server
  • a software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server
antivirus software
  • is used to detect and remove the presence of known viruses and other infections
cryptography
  • the study of sending and receiving messages securely in the presence of adversaries
public key encryption
  • involves techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure
  • A public-key encryption system involves the use of two keys. One key, known as the public key, is used to encrypt messages; the other key, known as the private key, is required to decrypt messages.
